This project presents an interactive Power BI dashboard built to analyze e-commerce sales data and generate meaningful business insights.
It helps track performance, understand customer behavior, and support data-driven decision-making.
Key Features
📈 Sales Overview – Total Amount, Profit, Quantity, and AOV
🛍️ Category Analysis – Clothing, Electronics, Furniture
💳 Payment Mode Insights – COD, UPI, Credit Card, EMI, Debit Card
🌍 State-wise Analysis – Sales & Profit distribution across states
📅 Monthly Profit Trends – Identify seasonal patterns
🧾 Sub-category Profit Analysis – Top & low-performing products

Key Outputs
Total sales reached 438K with a profit of 37K
Clothing category contributed the highest sales (~46%)
COD is the most preferred payment mode (~39%)
Maharashtra & Madhya Pradesh generated the highest sales and profit
Printers is the most profitable sub-category
Some months show negative profit, indicating loss periods
Average Order Value (AOV) is around 121K
Sales and profit show seasonal trends across months
