# 📊 E-Commerce Sales Dashboard (Power BI)

## 🚀 Project Overview
This project presents an **interactive Power BI dashboard** designed to analyze e-commerce sales data and generate meaningful business insights.  
It enables businesses to track performance, understand customer behavior, and make **data-driven decisions** effectively.

---

## ✨ Key Features

### 📈 Sales Overview
- Total Sales Amount  
- Profit  
- Quantity Sold  
- Average Order Value (AOV)  

### 🛍️ Category Analysis
- Clothing  
- Electronics  
- Furniture  

### 💳 Payment Mode Insights
- Cash on Delivery (COD)  
- UPI  
- Credit Card  
- EMI  
- Debit Card  

### 🌍 State-wise Analysis
- Sales and Profit distribution across different states  

### 📅 Monthly Profit Trends
- Identify seasonal patterns  
- Detect profit fluctuations  

### 🧾 Sub-category Profit Analysis
- Top-performing products  
- Low-performing products  

---

## 📊 Key Results & Metrics

| Metric | Value |
|------|------|
| 💰 Total Sales | **438K** |
| 📈 Total Profit | **37K** |
| 🛍️ Top Category | **Clothing (~46%)** |
| 💳 Most Used Payment | **COD (~39%)** |
| 🌍 Top States | **Maharashtra & Madhya Pradesh** |
| 🖨️ Most Profitable Sub-category | **Printers** |
| 📊 Average Order Value | **~121K** |

---

## 🔍 Insights

- Clothing dominates overall sales performance  
- COD is the most preferred payment method  
- Maharashtra leads in revenue generation  
- Some months show **negative profit**, indicating losses  
- Sales and profit follow **seasonal trends**
